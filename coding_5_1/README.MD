Ваша цель — внести изменения в полётное задание, добавив в него дополнительные
точки (waypoints) с другим параметром высоты и проверить в симуляторе работу кода
1. Установите приложение «Mission Planner»
2. Запустите симуляцию дрона из Mission Planner. Включите зеркалирование протокола Mavlink.
Для этого перейдите во вкладку «Simulation», нажмите «Multirotor» и выберите стабильную версию
симулятора. Зеркалирование Mavlink: CTRL+F → mavlink → TCP + отметить «write access» → start
3. Реализуйте в коде добавление двух дополнительных точек (waypoints),
высоту полёта укажите 50 метров. Запустите код.
4. Дождитесь пока аппарат перейдет в состояние «Ready to arm».
Состояние контролируется на вкладке «Data» в Mission planner.
5. Дайте команду «Arm». Дождитесь переключения аппарата из состояния Disarm в Arm.
Потом дайте команду «Takeoff» для взлета (команда в контекстном меню при нажатии
правой кнопки мышки).
6. Установите режим «Auto» на вкладке «Data» меню «Действие».
7. Убедитесь, что дрон выполняет загруженное вами полётное задание.
