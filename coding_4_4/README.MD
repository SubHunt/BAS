кода». Данное задание предназначено для отработки полученных навыков и
закрепления материала. Ваша работа не будет проверяться преподавателем, но вы
можете сами оценить себя сами по приведённому ниже чек-листу.
Описание задания: Графический интерфейс (GUI) уже собран , но логика работы

Цель задания: Найти места, отмеченные комментариями (Задание), и дописать
пропущенные логические условия, опираясь на правила безопасности БПЛА. Вам
нужно исправить 4 ошибки в коде. Не меняйте весь код, правьте только логические
блоки.

Задание:
Добавьте в условие if voltage < 3.5: изменение переменной check_result = False и
команду break, чтобы остановить проверку сразу после первой ошибки.

Задача 3. Запретить Взлет без ARM
Дрон может «взлететь» в программе, даже если статус DISARMED.
